Explore open-source public sector projects that are making a difference for British Columbians. Find something that inspires you and get involved.


EPIC

The Environmental Assessment Office's Project Information & Collaboration system makes it easier for the public and stakeholders to find important information and comment on environmental reviews. It has been released as a beta application – a fully-functional starting point upon which to build continuous improvements.

View Project

School Bus Inspection System Replacement Project

This project is modernizing the way vehicle inspectors conduct and track school bus inspections by decommissioning the existing mainframe application and replacing it with an Open Source solution that will help ensure the continued safety of the 3000+ school buses that are in operation in British Columbia today.

Housing Data Visualization Project

B.C. innovators and data visualization experts are creating prototoypes to demonstrate the value of government and other sources of data. In this example, the challenge is being applied to B.C. Housing market data.
