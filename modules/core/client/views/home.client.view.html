Code with us

We’ve been experimenting with a new way to hire developers to cut the code government needs. Instead of a bunch of paperwork, developers work directly in GitHub and get paid a fixed fee if their code meets the acceptance critera.

Read about what we’ve been learning and how we're evolving the model, or go directly to Open Opportunities to find work.
